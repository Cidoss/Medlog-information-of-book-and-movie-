## Medlog
Merupakan website yang memberikan informasi film dan buku sebelum kamu memilih untuk menonton atau membaca. website ini menyediakan data tiap buku dan film yang diambil dari TMDB untuk film dan Open Library untuk buku. website ini menyediakan fitur comment bagi user berpengalaman untuk sebagai pendukung tambahan informasi terkait buku atau film terkait.

## fitur utama
- Katalog Film: Menampilkan film populer dan filter berdasarkan genre.
- Katalog Buku: Menampilkan daftar buku berdasarkan genre
- Detail Info: Halaman detail untuk melihat informasi film dan buku.

## spesifikasi project
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Data Source: TMDB API & OpenLibrary API
